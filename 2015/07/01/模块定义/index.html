ES6 模块

  • import 导入
  • export 导出

import export

默认导出导出

  • 入import 变量名 from './xx.js'
  • 出export default 对象;
  • 声明式
    • 入 import {固定的导出key} from './xxx.js';
    • 出 export let 固定的导出key = 'xxx';
    • 或者两步导出
      • var 固定的导出key = 'xxxx';,
      • export {固定的导出key};
  • 全体导入
    • import * as 变量名 from './xxx.js';
    • 该对象 具备不同的key + default属性
  • import 和 export 只能在顶级作用域中使用，不能再{} 块级或者函数中使用
